JCC Website Prototyping Assignment
This project is part of the Human Computer Interaction course (COMP6800001). The objective is to create a 5-page responsive website using HTML, CSS, JavaScript, and Figma. The prototype consists of the following pages:

- Home Page
- Event Page
- Price List Page
- About Us Page
- Contact Us Page
  
Technologies Used:
- HTML
- CSS
- JavaScript
- Figma
